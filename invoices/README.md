# Invoices

Branded HTML invoices generated by the `/new-invoice` skill.

## How to create one

In Claude Code in this repo, type:

```
/new-invoice
```

Claude will ask for the client, date, and line items, then write a file here named `<invoice-number>-<client-slug>.html` (e.g. `194-alex-pym.html`) and open it in your browser.

## How to export to PDF

1. Open the HTML file in Chrome or Edge
2. **Ctrl + P**
3. Destination: **Save as PDF** (or "Microsoft Print to PDF")
4. Paper size: **Letter**
5. Margins: **None**
6. Options → **Background graphics: ON** ← important, otherwise the dark balance-due bar prints as a blank rectangle
7. Save

## Source files

- Template: `../templates/invoice.html`
- Skill: `../.claude/skills/new-invoice/SKILL.md`

If you want to change the brand treatment (logo, address, GST#, payment instructions), edit the template — the skill never modifies it.
