# Recaps

Branded HTML post-workshop recaps generated by the `/new-recap` skill.

## How to create one

After running an un-mission workshop:

1. On slide 33 of the deck, click **// export .txt** to download `capture-YYYY-MM-DD.txt` — the participant's words, exactly as you typed them in the room.
2. Move that file somewhere you can find it (anywhere is fine — `~/Downloads` works).
3. In Claude Code in this repo, type:

```
/new-recap
```

Claude will ask for the client name, the workshop date, and the path to the capture file. It will draft a set of themes from the captured words and let you tweak them inline before writing.

The output lands here as `<client-slug>-recap.html` (e.g. `alex-pym-recap.html`) and opens in your browser.

## How to export to PDF

1. Open the HTML file in Chrome or Edge
2. **Ctrl + P**
3. Destination: **Save as PDF**
4. Paper size: **Letter**
5. Margins: **None**
6. Options → **Background graphics: ON** ← important, otherwise the ink and red pages print blank
7. Save

## Source files

- Template: `../templates/recap.html`
- Skill: `../.claude/skills/new-recap/SKILL.md`
- Reference: `../examples/un-mission-workshop/alex-pym-recap.html` — the original recap the template was extracted from. Useful as a worked example of what a finished recap looks like.

If you want to change the brand treatment, body copy, or the fixed sections (premise, why-it-matters, examples, prompts, principles, what's-next, closing), edit the template — the skill never modifies it.
